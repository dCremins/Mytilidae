Wizards

A text adventure game built in Inform 7

Wizards was created with Inform and has IFID 056CA327-19B7-4D84-BC51-6A895F3D9760. To play a work like this one, you need an interpreter program: many are available, among them Zoom for Mac OS X and for Unix; Windows Frotz or Windows Glulxe for Windows. Or you can play without downloading anything by following the 'Play In-Browser' link, using the Quixe interpreter. You'll need to have Javascript enabled on your web browser.
